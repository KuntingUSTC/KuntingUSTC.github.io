如何认识、看待、应对新冠 ——总结自12月18日某一线医师针对新冠答患者问 一、 新冠的基础认知 （basis） 新冠不是大号感冒。他的特殊性表现在三方面：其一是容易加重基础病，其二是感染后的免疫屏障是三个月，也就是三个月内即使再感染，症状也轻微，但不会再长于三月。其三是三个月之后感染也存在比第一次感染还重的可能性。 新冠存在后遗症。当你得了之后体能和免疫水平还没完全恢复，一旦短期内再得，就一次比一次差，得了三四次就没有以后。所以还是尽量避免感染。 应对新冠既要做长期打算，又不要太害怕。目前疫情导致的重症率和死亡率依旧是非常低，该干嘛干嘛，但仍旧要预防好疾病，尽量避免感染。 新冠的主要表现为发高烧怕冷、咳嗽、喉咙痛这三关。他是风、寒、湿。 高烧了不用太过害怕。感染之后发到39度高烧的，往往恢复越快。始终温度上不去的反而恢复没这么快，恢复期较久。 为什么有基础疾病如心脑血管疾病、高血压等死亡率相对较高？因其血管较脆弱，发高烧时血压上去，就可能会脑溢血、心梗等。 二、如何预防新冠（prevention） 在可疑的环境里，一定戴好口罩、不揉眼睛、不摸鼻孔，手注意消毒。 进家门之前口罩扔掉，有条件在户外把鞋脱下，并在门口地毯上喷消毒液，鞋底浸一下，外套挂在外面，然后消毒、洗手。 房间里的空气消毒，可以关起门来熏艾条。具体而言，门窗密闭，点燃艾条之后人离开，十几二十分钟后开门窗通风。 家里已经有人感染，其他人包括患者的人都要戴口罩。高病毒载量感染和低载量感染获得的抗体是一样的，但高载量感染容易引发重症。因此就算家里有人得了，还是要做好防护，争取被感染的时候病毒越少越好。 家里有人阳转阴，记得把地板、家具等带消毒剂抹布擦一遍。含氯消毒剂、酒精都有效。 藿香正气水可以作预防用，并且在你吃错药了又拉又吐的时候也可以作治疗用。 优先考虑居家。家里面有老人、小孩子的准备两个东西。一个是家用制氧机，这是在家和在医院最大的区别。第二个是医用氧饱和夹，夹手指上测氧饱和度。 三、 得新冠时注意的点（major points） 得新冠，最重要的第一件事是躺平，即卧床休息。前面提到，新冠致死的很大原因是发烧工作，过度劳累带来的病毒性心肌炎。只要觉得乏力，就躺平。 千万别乱吃药。如果身体健康没有基础病最多吃点退烧药就可以。成年人38度吃退烧药，小孩子39度吃退烧药。 发烧之后怕冷多盖被子，喝些热水、热姜汤，血糖没问题就上红糖水，加上退烧药更容易发汗。退热的自然规律是发热到达极点后往下走，先高后低，因此吃退烧药后温度没有一下下来是正常，不要惊慌。不要轻易加量、频繁服用。一般规律是三个小时没有效果再吃，一天吃个三次最多。 喝热水，根据自己的需求喝，不要喝到自己想吐。一个白天三升的水足够。 维C是非常好的。其能减轻炎症风暴，缓解咽喉疼痛。注意是那种两块钱一瓶的不含糖维C，每天最多吃到十片，记得要含到溶解。记得不用泡腾片，因为泡腾片有糖，而咽喉炎要限糖，不然可能加重病情。青柠檬也不错，柠檬水加蜂蜜对咽喉痛有用。 再次强调，千万不要乱吃药。如果没有准备药，且没有基础疾病，年纪不大，营养水平还可以的，越简单越好，吃个退烧药，三天内也能扛过去。新冠治疗符合一般病毒性感冒规律。 对于普通人，病患上之后高蛋白、高热量、好消化饮食，不要高脂肪，会加重身体负担。如果平时没有糖尿病、过度肥胖就这样做。粥、面条这些发酵的食物是首选，烹调方式尽量简单，不要吃炒菜、爆炒的食物，这些会加重炎症反应。 对于基础病患者。高血糖患者要备好降血糖的药，阳了之后血糖极易上升，特别注意检测与饮食。高血压患者阳了之后发烧，血压也会比以往更高，降压药也一起吃，不要考虑有相互作用，控制住血压。 对于甲亢和甲减患者。甲亢患者很容易超高热，要小心，退烧药要更足。甲减患者烧的时间会长，别人七天恢复你可能十四天。 如果出现心肌炎、呼吸衰竭等症状再去医院。不然躺平更好，排队耗体能，尽量自己克服。 阳了以后房间必须每天消毒。不主张向空气中喷酒精，含氯消毒剂腐蚀性太强。如果没有物品，每天十分钟开门通风换气，不会导致病毒往外传播。煮白醋首选，或姜或者大蒜煮一下，或艾条点燃人离开熏房间。紫外灯消毒也是很有用的（人不要在下面照射）。 咳嗽为次要症状。买一些生姜就能很好的应对。 买点小黄姜、小葱、冰糖备着。姜葱白汤本身就可以退烧。 即使是有反复发热，最高的温度会逐渐下降，不用太担心。不主张在刚刚温度下来一点的时候去洗澡。 三、 一些次要的点（minor points） 基本能打疫苗的还是建议打，相比疾病风险，疫苗的风险是可以承受的。有基础病的应该是值得冒险去打疫苗，比患新冠的风险小。 所有退热药、感冒药都可以吃，不一定要对乙酰氨基酚，布洛芬等都可以，小心过敏。实在不行，生姜葱白汤，喝到出汗为止，再不行就喝热水，都有用。 新冠最可怕的并发症是心肌炎，可千万要躺平。该吃吃该喝喝，关键是要休息。新冠可以喝点小酒是有好处的。 你刚刚有症状去做抗原可能假阴性。明后天测可能阳了。普通流感和新冠病毒的感染无法区分，目前要做筛查才能区分，从症状上完全没法区分。 阳了之后转阴还是会再阳。一般来说阳过一次三个月内感染症状会很轻，三个月后又可能会感染。阳过以后但凡觉得累、疲惫、后遗症状有，赶紧休息。 感染后胃口不好正常，会慢慢恢复。 每一次都带着最好的状态去面对新冠，你才是安全的。一定要注意防护，以及之后的休息。 不要心存侥幸，尽量晚阳、低病毒载量的阳。阳了后尽量躺平休息，不要折腾。 ——最后祝大家远离大阳间，守住小阴间，阳间不值得！
